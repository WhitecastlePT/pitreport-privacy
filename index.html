02

Permissões Utilizadas e Finalidade

A aplicação solicita as seguintes permissões do dispositivo, cada uma com uma finalidade específica e justificada:

📷
CAMERA
Permite fotografar problemas urbanos (buracos na estrada, danos em infraestruturas) para anexar às denúncias submetidas.
📍
ACCESS_FINE_LOCATION
ACCESS_COARSE_LOCATION
Obtém as coordenadas GPS do dispositivo para geolocalizar automaticamente cada denúncia submetida pelo utilizador.
🎙️
RECORD_AUDIO
Mede os níveis de ruído ambiente (dB) para a funcionalidade de denúncia de poluição sonora urbana.
🖼️
READ_MEDIA_IMAGES
Permite selecionar imagens existentes na galeria do dispositivo para anexar a denúncias, sem que as imagens sejam lidas fora desse contexto.
03

Dados Recolhidos

A aplicação pode recolher os seguintes dados, apenas quando o utilizador submete uma denúncia de forma voluntária:

  • Fotografias tiradas ou selecionadas pelo utilizador
  • Coordenadas GPS no momento da denúncia
  • Nível de ruído medido (em decibéis), quando aplicável
  • Descrição textual da ocorrência (introduzida pelo utilizador)
  • Data e hora de submissão
Importante: A aplicação não recolhe dados de forma passiva ou em segundo plano. Nenhum dado é partilhado com terceiros para fins comerciais ou publicitários. A câmara e o microfone são acedidos apenas quando o utilizador inicia ativamente as respetivas funcionalidades.
04

Armazenamento e Segurança

Todos os dados submetidos são armazenados em Google Firebase (Firestore Database e Firebase Storage), uma plataforma cloud gerida pela Google LLC, com servidores localizados na União Europeia.

  • A transmissão de dados é cifrada via HTTPS/TLS
  • O acesso à base de dados é controlado por regras de segurança do Firebase
  • Não é realizado nenhum processamento de imagem ou áudio fora do dispositivo além do armazenamento
  • Os dados são conservados pelo período necessário para fins académicos do projeto
05

Partilha de Dados com Terceiros

Os dados recolhidos não são vendidos, alugados nem partilhados com terceiros para fins comerciais.

A única entidade que processa os dados é o Google Firebase, como infraestrutura de armazenamento, sujeita à Política de Privacidade da Google.

06

Direitos dos Utilizadores

Em conformidade com o Regulamento Geral sobre a Proteção de Dados (RGPD — Regulamento UE 2016/679), os utilizadores têm direito a:

  • Acesso — solicitar uma cópia dos dados pessoais armazenados
  • Retificação — corrigir dados incorretos ou incompletos
  • Eliminação — solicitar a remoção dos seus dados ("direito ao esquecimento")
  • Oposição — opor-se ao tratamento dos dados em qualquer momento
  • Portabilidade — receber os dados num formato estruturado e legível por máquina

Para exercer qualquer um destes direitos, contacte o responsável através do endereço indicado na secção seguinte.
